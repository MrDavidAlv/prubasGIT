GIT

Crea un repositorio
>> git init

comando para ver el estado del repositorio  archivos o carpetas
>> git status

Agregar archivos para hacer un commit
>> git add <file>

Agregar todos los archivos creados, modificados o elminados
>> git add .

quitar un archivo que se agrego con git add
>> git rm --cached <file>

crear un commit con los archivos o directorios aÃ±adidos
>> git commit -m "primer commit pruebas con GIT"

crear una rama y pasarse a ella
>> git checkout -b DevMario

cambiarse de una rama a otra
>> git checkout NombreRama

para hacer un merge o fusion de cambios
>> git merge NombreRama

muestra toda la informacion del repositorio
>> git log

muestra toda la info del repositorio con una interfaz grafica
>> gitk
